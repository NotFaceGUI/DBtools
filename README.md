# DBtools
一个小巧快速编辑表数据的工具，使用Swing绘制，原生JDBC来操作数据库，通过可视化界面操作数据，支持修改、创建、删除表和数据等等基本功能。同时支持导出sql结构和数据，以及excel文件，具有一定的学习意义。
